# Moose Antler 1.3.0

- <Tag type="new">New</Tag> **Settings Page**: A comprehensive settings page has been added.  
- <Tag type="improvement">Improved</Tag> **Persistence**: The app now remembers all your settings on restart, including:  
  - Light/Dark mode preference  
  - Primary color theme  
  - Startup action (open last file or new canvas)  
  - Last used window size and position  
  - Last used mind map layout (vertical/horizontal)  
  - Last used zoom level  
- <Tag type="new">New</Tag> **Zoom Slider**: A slider has been added to the canvas for fine-grained zoom control.  
- <Tag type="new">New</Tag> **Legal & About**: Added *Privacy Policy*, *Terms & Conditions*, and a dynamic *About* section with app info.  
- <Tag type="new">New</Tag> **UI**: Reordered AppBar icons for a more logical and intuitive layout.  
- <Tag type="bugfix">Bugfix</Tag> Node text is now always readable in dark mode.  
- <Tag type="bugfix">Bugfix</Tag> Saving a file now correctly adds the `.moose` extension if the user forgets.  
- <Tag type="bugfix">Bugfix</Tag> The Windows installer now correctly installs to `C:\Program Files` for 64-bit systems.  
- <Tag type="improvement">Improved</Tag> Installer now bundles the required Microsoft VC++ Redistributable to prevent DLL errors on clean systems.  
- <Tag type="improvement">Improved</Tag> Application icon and branding updates.
